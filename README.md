# 基于百度AI的情感分析微信小程序

#### 介绍
实现微信一键登录；    
用来存储用户每天编写的文字记录；    
通过用户提交的文本记录、分析个人的情绪倾向并记录下来；   
统计和查看用户每次记录下来的情绪结果；   
根据用户近期的情绪表现，随机提示一些管理情绪的建议；
